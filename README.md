# Agency Talent

A collection of AI agent talents converted from [agency-agents](https://github.com/msitarzewski/agency-agents).

Contains **145** talents across the following categories:

- **design** (Designer)
- **engineering** (Engineer)
- **game-development** (Game Developer)
- **integrations** (Engineer)
- **marketing** (Marketer)
- **paid-media** (Marketer)
- **product** (Product Manager)
- **project-management** (Project Manager)
- **sales** (Sales)
- **spatial-computing** (Engineer)
- **specialized** (Specialist)
- **support** (Support)
- **testing** (QA Engineer)

## Usage

Register this repo URL in Talent Market to make all talents available.
Each talent directory contains:

```
talent-name/
  profile.yaml    # Agent identity & configuration
  skills/core.md  # Agent skills & instructions
  tools/manifest.yaml  # Tool declarations
```
