## Launch checklist (next 24 hours)

1. **Lock handles manually** — try-to-register `@midtierlegends` on TikTok, Instagram, YouTube, X, Threads, Beacons, Linktree. Screenshot each. If primary is taken on TikTok / IG / YouTube, pause and re-evaluate.
2. **Buy domains** — `midtierlegends.com` + `.co` + `.tv` + `.net` defensive (~$60).
3. **File USPTO intent-to-use** in classes 41 (entertainment), 25 (apparel), 9 (downloadable content). ~$1,050–$1,400. File before first public post.
4. **Never put "NBA" in the name, handle, or trademark application.**
5. **Brief the illustrator** with [`identity-package.md`](./identity-package.md) — visuals carry the reverent tone that the word "mid" alone can't.
