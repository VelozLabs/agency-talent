**Agent**: Narrative Designer (`narrative-designer`)
**Input**: C-00 brief, C-01 brand identity, Phase 1 Editorial Filter (session-09, Loop 5)

## Agent Directive

You are building the copy architecture, not writing individual strings. Your job: define the system that produces every piece of text in the app — the rules, the voice pillars, the structural patterns — so that anyone writing copy later can produce strings that sound like the same product.
