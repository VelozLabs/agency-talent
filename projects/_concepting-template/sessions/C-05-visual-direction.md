# Session C-05 — Visual Mockup Direction
**Agent**: Visual Storyteller (`design-visual-storyteller`)
**Input**: C-01 brand identity, C-02 IA, C-03 design system, C-04 voice system
**Phase**: 2 — Concepting

---

## Agent Directive

You are not designing screens — C-03 did that. You are telling the story of what the product looks and feels like. Your output is the narrative that guides the image generation (C-06) and the prototype (C-07). Think: if this product had a 60-second App Store preview video, what would happen in it?

---

## Deliverables

### 1. Key Moment Storyboard

Identify 5 key moments in the user experience. For each:

```
Moment: [PLACEHOLDER — name]
Context: [PLACEHOLDER — what is the user doing/feeling]

What the user does: [PLACEHOLDER — gesture/action]
What changes: [PLACEHOLDER — how the screen responds]
Emotional beat: [PLACEHOLDER — what the user feels at this moment]
```

### 2. Visual Mood Direction

Describe the visual mood in sensory terms — not "minimalist" or "clean" but specific:

```
Light quality:    [PLACEHOLDER — e.g., "warm indirect, like late afternoon through frosted glass"]

```

### 3. App Store Narrative

### 4. What This Product Is NOT

